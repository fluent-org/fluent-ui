## API

```
import Progress from '@fluent-windows/core/Progress'
// or
import { Progress } from '@fluent-windows/core'
```

### Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| value | number |  | 当前完成的百分比，必须为数字且介于 0 到 100 之间。如果不设置，将展示循环的动画。 |
